Request And Background Flows

Request path invariant

The client waits only for local durability. S3 and ClickHouse are deliberately off the synchronous ingest path.

Raw data source

S3 stores immutable NDJSON parts. ClickHouse rows keep byte pointers back to exact accepted raw lines.

Tenant scoping

The server stamps organization identity into events and scopes read queries against tenant/organization IDs.
